# KavyMeet – Secure Video Meeting App

**Original Project by Kavy Soni**

KavyMeet is a branded browser-based video meeting app made for GitHub and Vercel deployment. It includes room generation, lobby page, camera preview, meeting room, mic/camera controls, screen sharing, invite link copy, local room chat, and visible ownership branding.

## Features

- Landing page inspired by a modern video meet UI
- Working **Generate Room** button
- Lobby with email/name/room number fields
- Camera and microphone preview
- Meeting room with local video stream
- Mic mute/unmute and camera on/off controls
- Screen sharing support
- Copy invite link
- Local room chat
- Fully responsive design
- Kavy Soni branding and ownership notice
- Static project: no Next.js version issue, no build error, no API key needed

## Deploy on Vercel

1. Upload this folder to GitHub.
2. Go to Vercel → New Project.
3. Import the GitHub repository.
4. Framework Preset: **Other** or **No Framework**.
5. Build Command: leave empty.
6. Output Directory: leave empty.
7. Deploy.

## Important Note

This stable version avoids Jitsi/third-party moderator errors. It is made as a polished frontend/video-meeting portfolio project. Real multi-user peer-to-peer video requires a signaling server or a video provider account.
